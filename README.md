# nunjucks-tailwind-starter

A basic frontend setup with Gulp, Tailwind CSS and Nunjucks.

## Project setup

NPM

```
npm install --global gulp-cli
npm install

```

Yarn

```
yarn global add gulp-cli
yarn
```

### Start and hot-reloads for development

```
yarn start
// or
npm start
```

### Build for production

```
yarn build
// or
npm run build
```
